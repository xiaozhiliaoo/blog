# 合同看的不够仔细

看关键点，责任和权利对等。

# 没有争取到有利于买方的合同

虽然谈了很多，细节确认了很多，但是其实并没有签一个买房合同

# 违约责任不明确

风险规避，首先要知道可能得风险。

# 价格谈的不够坚定

心理战。

# 中介费还可以再低

开口3%，谈到2.85%，实际2.5%。可以找小中介。

# 家居留什么没有仔细确认

需要写到合同上。

# 首付交的太早

等业主解完抵押在交，几十万存短期/余额宝也有不少利息。

# 其余整理

交易方式：自行成交，中介成交，

买房流程：市公积金，省公积金，全款

谈价时间：和业主砍价，中介费，

贷款手续费：可以不交，打贝壳中介。

拆解合同思路：数字，违约事项。

案例分析：

买：全款，纯公积金，组合贷

卖：徐乐

商贷提前还款方式，时间，金额等。是否可以缩短期限，或者是缩短金额。一年内可以提前还。

公积金贷款提前还款方式，时间，金额等，是否可以缩短期限，或者是缩短金额。半年内可以提前还。

贝壳品质监督：10106188
